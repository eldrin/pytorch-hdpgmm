hdpgmm.parameters Module

class hdpgmm.parameters.Parameters

It is a base dataclass that is inherited to diverse parameter objects.

for some of the intermediate values (i.e., Eq_eta_1, Eq_eta_2, etc.), We followed (Blei and Jordan 2006) and Wikipedia entry of the exponential family.

Parameters

Returns
