hdpgmm.math.mat_sqrt(A: Tensor, method: str = 'cholesky', ensure_pos_semidef: bool = True, eps: float = 1e-06) → tuple[torch.Tensor, torch.Tensor, torch.bool, torch.float32]

Compute “square root” of the given symmetric matrix

It computes the “square root” of the given symmetric matrix. It provides several methods, including the Cholesky decomposition (default), SVD, and finally eigen decomposition. The implementation is based on here.

Parameters

  • A – input real symmetric matrix

  • method – specifying which method is used. {‘cholesky’, ‘svd’, ‘eig’}

  • ensure_pos_semidef – if True, it tries to fix the issue by some tricks. however most of the case it should be used very carefully otherwise it’ll just produce a degenerated outcome

Returns

the input matrix A, which could be reconstructed from adjusted square root matrix of it L: the decomposition of the input matrix which by computing the dot product

of itself can recover the input matrix

isposdef: boolean indicate the given matrix is positive semi definite abslogdet: absolute log determinant computed from torch.linalg.slogdet

Return type

A

Raises

ValueError if method is not supported –
